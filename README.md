# java-ladder
사다리타기 미션 저장소

### 기능 목록
1. 입력
    - [x] player, reward 입력 받기
    - [x] 사다리 높이 입력 받기
    - [x] 글자 수 제한
    - [x] 비어 있는 이름
    - [x] 중복된 이름 처리
    - [x] player 수 == reward 수
2. 출력
    - [x] 그려진 사다리 출력
3. 사다리타기 
    - [x] 그릴 수 있는 사다리 생성하기
    - [x] 사다리를 탔을 때 결과 구하기
        - [x] 플레이어 입력 -> 리워드 출력
